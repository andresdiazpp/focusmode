# FocusMode

App de menu bar para Mac que bloquea sitios web distractores.

## Qué hace hoy

**DeadZone** — bloquea sitios permanentemente hasta que los desactives manualmente. Útil para Instagram, TikTok, o cualquier sitio que no quieres ver en absoluto.

**VoidList** — bloquea sitios por un tiempo específico. Puedes elegir una duración (ej. 2h 30m) o una fecha y hora exacta. Cuando el tiempo termina, el bloqueo se levanta solo — aunque la app esté cerrada.

El contador de tiempo restante aparece en el menu bar en tiempo real.

## Cómo correrlo

```
python3 main.py
```

## Requisitos

- macOS
- Python 3
- `rumps` — `pip install rumps`
